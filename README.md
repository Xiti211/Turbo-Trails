# Turbo-Trails
Бесконечные дороги, острые повороты, адреналин в крови! Управляй автомобилем с помощью кнопок, преодолевай горы, мосты и городские районы. Следи за топливом, собирай монеты, избегай аварий. В магазине — множество машин: открывай их за монеты или покупай через донат. Побей свой рекорд, прокачай автопарк! Простое управление, ярка графика
